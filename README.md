# CodeAlpha_Iris_Classification
🌸 Iris Flower Classification using Machine Learning
# 📌 Project Overview
This project is based on Iris Flower Classification, where a machine learning model is used to predict the species of an iris flower based on its physical measurements.
The dataset contains features like sepal length, sepal width, petal length, and petal width, which are used to classify flowers into three categories:
Setosa
Versicolor
Virginica

# 🎯 Objective
The main objective of this project is to:
Build a classification model using machine learning
Predict the correct species of iris flowers
Understand data preprocessing, visualization, and model evaluation

# 🛠️ Tech Stack
# Language: Python
# Libraries:
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

# ⚙️ Workflow
🔹 Data Preprocessing
Cleaned column names
Removed unnecessary columns (ID)
Checked for missing values
🔹 Exploratory Data Analysis (EDA)
Pairplot for feature relationships
Heatmap for correlation analysis
🔹 Model Building
Implemented Random Forest Classifier
Split dataset into training and testing sets
🔹 Model Evaluation
Accuracy Score
Classification Report
Confusion Matrix

# 📈 Results
Achieved ~95%+ accuracy
Model performs effectively in classifying flower species
